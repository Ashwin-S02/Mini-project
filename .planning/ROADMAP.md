# Roadmap

## Phase 1: Lead Scoring Algorithm
Implement a scoring system to prioritize leads based on their digital presence and service needs.

### Success Criteria:
- Algorithm defined and agreed upon.
- Scoring logic integrated into the scraper.
- Scores added as a new column in Google Sheets.

## Phase 2: Deployment
Deploy the Flask application to a free hosting provider (e.g., Render) to allow client review.

### Success Criteria:
- Application accessible via a public URL.
- Environment variables (Google Sheets creds) securely configured.
- Playwright/Chromium dependencies correctly handled (Docker).

## Phase 3: CRM Integration
Directly push leads to CRM via API or webhooks.
